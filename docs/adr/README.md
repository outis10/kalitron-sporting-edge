# Architecture Decision Records

ADRs capture decisions arquitectónicas importantes — el *qué* se decidió, el *porqué*, y las consecuencias.

## Formato

Cada ADR sigue la plantilla de Michael Nygard (simplificada):

```
# ADR-NNN: Título
Fecha: YYYY-MM-DD
Estado: Propuesto | Aceptado | Deprecado | Reemplazado por ADR-NNN
```
